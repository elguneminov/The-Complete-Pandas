Welcome to the Section Time Series Advanced / Financial Time Series


In this Section, we are going to import, manipulate and analyze stock price data.

Before we can start, we have to install one additional package to our Anaconda Distribution:

  • yfinance

With this additional package, we are able to import stock price data from Yahoo Finance.


To install, please open the Anaconda Prompt (Terminal on macOS or Linux) and enter the following command:

  • pip install yfinance (press y when asked)


Before, you should make sure that you have the most recent Anaconda Version installed. Use the command:

  • conda update anaconda (press y when asked)
